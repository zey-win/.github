# ZeyWin

ZeyWin builds and releases mobile game SDK infrastructure:
Unity Ads SDK, CrashGuard SDK, private game sources,
and a public GitHub Actions APK releaser.

## OS
- Ubuntu 22.04

## Soft
- Unity 6+

## Builds
- android

## Core Repositories

- `ZeyWinAdsSDK-Unity` - monetization SDK, startup gating, UniWebView, native banners, AdMob fallback.
- `CrashGuardSDK-Unity` - crash/runtime guard SDK for Unity mobile releases.
- `game` - private Unity game source used by release workflows.
- `ci-cd` - public APK releaser and reporting workspace.
